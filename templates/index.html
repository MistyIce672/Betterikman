Better Ikman

Better ikman is made for quick ikman searches
Better ikman has easily accesible filters
and loads all results to on page.
Better ikman is multi threaded for faster seacrhes

How to use

Multi search: adding a comma in bettween search terms will
search for differnt terms and merge results
multi search is multithreaded where each term has its own thread
but since a single term only has a single thread getting many pages can
take time.

Single Search: single searches are multi threaded to load faster
if ur search term returns large amount of results runing it single
search mode will get ur results faster
